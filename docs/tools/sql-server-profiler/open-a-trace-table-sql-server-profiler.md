# Open a trace table (SQL Server Profiler)

[!INCLUDE [SQL Server Azure SQL Managed Instance](../../includes/applies-to-version/sql-asdbmi.md)]

This article describes how to open an existing trace table by using [!INCLUDE [ssSqlProfiler](../../includes/sssqlprofiler-md.md)].

## Open a trace table

1. On the **File** menu, point to **Open**, and then select **Trace Table**.

1. Connect to the server instance that contains the trace table.

1. In the **Source Table** dialog box select the database that contains the saved trace table from the **Database** list.

1. From the **Table** list, select the table name. Select **OK**.
