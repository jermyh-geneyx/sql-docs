# Open a trace file (SQL Server Profiler)

[!INCLUDE [SQL Server Azure SQL Managed Instance](../../includes/applies-to-version/sql-asdbmi.md)]

This article describes how to open a trace file by using [!INCLUDE [ssSqlProfiler](../../includes/sssqlprofiler-md.md)].

## Open a trace file

1. On the **File** menu, select **Open**, and then select **Trace File**.

1. In the **Open File** dialog box, select the trace data file you want to open.
