# Close a trace window (SQL Server Profiler)

[!INCLUDE [SQL Server Azure SQL Managed Instance](../../includes/applies-to-version/sql-asdbmi.md)]

This article describes how to close the trace window for traces that are running or for saved traces you're viewing by using [!INCLUDE [ssSqlProfiler](../../includes/sssqlprofiler-md.md)].

## Close a trace window

1. Select the trace window you want to close.

1. On the **File** menu, select **Close**.

> [!IMPORTANT]  
> If the trace is running, closing a trace window stops the associated trace.
