# Keyboard shortcuts for SQL Server Unit Test Designer

You can use the following keyboard shortcuts to navigate within the SQL Server Unit Test Designer.

| Action or Menu Option | Keyboard Shortcut |
| --- | --- |
| Move from the Transact-SQL editor to the navigation bar. | <kbd>Ctrl</kbd>+<kbd>F2</kbd> |
| Move forward through navigation bar elements. | <kbd>Tab</kbd> |
| Move backward through navigation bar elements. | <kbd>Shift</kbd>+<kbd>Tab</kbd> |
| Move between different panes in the Database Unit Test Designer.<br /><br />**Note:** This shortcut doesn't work if you specify C# for your developer profile the first time that you open Visual Studio. | <kbd>F6</kbd> / <kbd>Shift</kbd>+<kbd>F6</kbd> |
